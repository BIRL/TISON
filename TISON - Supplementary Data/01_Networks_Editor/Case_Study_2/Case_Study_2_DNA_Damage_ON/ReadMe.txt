The cell fate logic and network has been adapted from the work of Choi et al. (Reference: Choi M, Shi J, Jung SH, Chen X, Cho K. Attractor Landscape Analysis Reveals Feedback Loops in the p53 Network That Control the Cellular Response to DNA Damage. 2012;5(November). 

Folders/Files:-
Description: Contains the network type, analysis type and network name

Network: Contains the network for this case study which is a weight based p53 Network with DNA damage ON, case study with the first half represent the basal values and the second part contains the interactions 

Parameters: Contain the Search Duration, Sampling Rate and Heuristic values used for this network analysis (if applicable)

Sample_Results: Contain the sample Deterministic Results for this analysis using TISON's network editor

Cell_Fate_Classification: An excel file (.csv) file which has the cell fates associated with the nodes as given in the network of Choi et al to study p53 Network in MCF-7 cell lines.
